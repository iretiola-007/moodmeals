A brief history of HTML

HTML was created by Tim Berners-Lee in 1991 to share documents over the World Wide Web. It has evolved through multiple versions, with HTML5 being the current standard.

HTML stands for HyperText Markup Language, and it's not a programming language - it's a markup language used to structure content on the web.

Basic Structure Tags

<!DOCTYPE>
This tag tells the browser what version of HTML to expect and is placed at the very first line of any .html file to avoid rendering issues or unexpected behaviour. Note: This tag isn't an HTML element, it's a declaration and for this reason, it doesn't have a closing tag.
<html>
This defines the root element that wraps all HTML content. Any HTML code you write should go in this element. This element has an opening and a closing tag.
<head>
This tag contains the metadata, links to stylesheets, and the page title.
<title>
This sets the title shown on the browser tab.
<body>
This holds all the visible content on a webpage, such as headings, paragraphs, images, and more.
<h1> to <h6>
Heading tags, with <h1> as the largest and <h6> as the smallest.
